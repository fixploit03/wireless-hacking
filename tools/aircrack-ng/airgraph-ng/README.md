# Airgraph-NG

#### Membuat grafik CAPR:

```
airgraph-ng -o [output] -i [file_csv] -g CAPR
```

Grafik yang dihasilkan yaitu grafik yang menampilkan hubungan antara client dengan AP yang terhubung. Grafik ini berguna untuk memvisualisasikan perangkat mana saja yang terasosiasi dengan AP tertentu.

#### Membuat grafik CPG:

```
airgraph-ng -o [output] -i [file_csv] -g CPG
```

Grafik yang dihasilkan yaitu grafik yang menampilkan probe request umum dari berbagai client untuk mengidentifikasi jaringan WiFi yang sering dicari.
