# hostapd-mana

Homepage: [https://github.com/sensepost/hostapd-mana](https://github.com/sensepost/hostapd-mana)

## Cara Instal

```
sudo apt-get update
sudo apt-get install hostapd-mana
```

## Cara Menggunakan

#### Penggunaan dasar:

```
sudo hostapd-mana [config_file]
```

#### Mode debug:

```
sudo hostapd-mana [config_file] -d
```

#### Jalankan di background:

```
sudo hostapd-mana [config_file] -B
```
